# Estrategia Multi-Activo Colombia 2025-2030

Este repositorio contiene el código fuente y los datos utilizados para el informe final de portafolio presentado en noviembre de 2025.

## Autores
* Gianmarco Pernett
* Gabriel Moreno
* Esteban Ramírez
* Sebastian Baquero

## Contenido del Repositorio
1. **Analisis_Portafolio.ipynb**: Notebook de Python con:
   * Descarga de datos de Yahoo Finance.
   * Procesamiento de tasas TES (Yield-to-Price).
   * Simulación de Monte Carlo y Frontera Eficiente.
2. **tasas_banrep.xlsx**: Datos históricos de las curvas cero cupón (Fuente: Banco de la República).

## Instrucciones de Replicabilidad
Para ejecutar este código:
1. Abra el archivo `.ipynb` en Google Colab.
2. Suba el archivo `tasas_banrep.xlsx` al entorno de ejecución de Colab.
3. Ejecute todas las celdas en orden.
